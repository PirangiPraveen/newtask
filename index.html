Visual Studio Code (VS CODE):-

Visual Studio Code, also commonly referred to as VS Code, is a source-code editor developed by Microsoft for web browsers, Windows, Linux and macOS. Features include support for debugging, syntax highlighting, intelligent code completion, snippets, code refactoring, and embedded version control with Git.
